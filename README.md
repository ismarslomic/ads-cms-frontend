# Accenture Digital Signage Frontend

## Prerequisites
NodeJS
npm
bower
grunt
[Heroku Toolbelt](https://toolbelt.heroku.com/)


## Getting started
```
git clone https://github.com/ismarslomic/ads-cms-frontend.git
cd ads-cms-frontend
heroku git:clone -a 
npm install
grunt build
grunt build:heroku
grunt deploy:heroku
```
